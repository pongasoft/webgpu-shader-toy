WebGPU Shader Toy is a free tool for the WebGPU Shading Language (WGSL).

This tool is using emscripten (web assembly) for the compiler and ImGui for the GUI with emscripten-glfw for the backend and WebGPU for the renderer.

This tool has no tracking, ads or server-side component: it is running 100% in your browser. It uses localStorage to store information across browser sessions.

It has been inspired by the popular project Shadertoy.
